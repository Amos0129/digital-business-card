#### application-dev.properties
開發時使用的配置檔, 請替換下列配置
* DEV_RUN_PORT
* DB_HOST_IP
* DB_PORT
* DB_NAME
* COMMAND_USER
* COMMAND_USER_PWD
* QUERY_USER
* QUERY_USER_PWD

- - - 
## Server Setting
請依據伺服器上定義的 jndi 進行配置
* JNDI_COMMAND_NAME
* JNDI_QUERY_NAME
